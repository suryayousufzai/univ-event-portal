# Backend - University Event Registration Portal

## Technology Stack
- Node.js / Express.js (or your chosen backend)
- MySQL Database
- JWT Authentication
- Payment Gateway Integration (Sandbox)

## Project Structure
- `/routes` - API route handlers
- `/models` - Database models
- `/controllers` - Business logic
- `/middleware` - Authentication, validation

## Database
See `/docs/design/backend-schema.md` for complete schema.

## API Endpoints
See `api-endpoints.md` for all endpoints.

## Setup Instructions
1. Install dependencies: `npm install`
2. Configure database connection
3. Run migrations
4. Start server: `npm start`
