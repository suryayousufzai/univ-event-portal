# Frontend - University Event Registration Portal

## Technology Stack
- React.js
- Tailwind CSS
- React Router
- Axios (for API calls)

## Project Structure
- `/src/components` - Reusable components
- `/src/pages` - Page components
- `/src/services` - API service layer
- `/src/styles` - CSS and Tailwind configs

## Pages Implemented in Sprint 1
- Login Page
- Registration Page  
- Events List Page

## Design
See Figma design: https://audio-task-68756584.figma.site/
Screenshots available in `/frontend/screenshots/`

## Setup Instructions
1. Install dependencies: `npm install`
2. Configure API base URL
3. Start development server: `npm start`
